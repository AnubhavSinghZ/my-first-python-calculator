# my-first-python-calculator
A simple calculator using Python while loop
This is my very first Python project! I built this to practice logic using 'while' loops and conditional statements.

## How it works
-It takes two number from the user 
-It perform Addition, Subtraction, Multiplication, and Division 
-The program runs continuously until the user types 'exit'.

## Tech used 
LANGUAGE- Python 3
CONCEPTS- while loop, if-else logic, input handling
